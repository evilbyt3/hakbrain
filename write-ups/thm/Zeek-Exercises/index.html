# Phising

Looking @ the generated http.log reveals that 3 files we’re retrieved, from which 2 seem suspicious (.exe & .doc):

I took note of the ip & domain in defanged format: 10[.]6[.]27[.]102, smart-fax[.]com. Going further, we can make use of Zeek’s scripting language to extract the files & get the hashes for the files:

With this information we can plug the md5 hash to virustotal in order to get a better idea of what we’re up against. By skimming through the info there, we learn that the doc file is most probably decepting the user into allowing VBA to run which allows it to drop an exe file.

Since we have also extracted the knr.exe file we can look into that as well. Commonly seen as PleaseWaitWindow.exe is most probably a variant of NetWire. One of the contacted domain names is hopto[.]org
